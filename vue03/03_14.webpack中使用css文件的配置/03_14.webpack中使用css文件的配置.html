安装css-loader:npm install --save-dev css-loader npm install style-loader --save-dev

webpack打包css报错（已解决）：UnhandledPromiseRejectionWarning: TypeError: this.getResolve is not a function

原因是 css-loader 和 style-loader 版本过高

解决方案：

https://blog.csdn.net/qq_43377853/article/details/108485499

package.json里手动下调了一下版本

"css-loader": "^3.3.0", "style-loader": "^1.0.0",

然后再命令行中 依次输入：

npm install

npm dev
